---
title: Reset Password
description: "Reset your password to access your Dokploy account."
---


To reset your password, follow these steps:

1. Log in in to your VPS.
2. Run 
to get the container ID of the dokploy container.

```bash
docker ps
```
3. Run command below to open a shell in the dokploy container.
```bash
 docker exec -it <container-id> bash -c "pnpm run reset-password"
``` 
 
4. It will display a random password. Copy it and use it to access again to the dashboard.
