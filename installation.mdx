### Requirements 

To have a nice experience with Dokploy, it is necessary to have at least 2GB of Ram and 30GB of Disk, because Dokploy runs with docker, and when docker run the build it consume more resources so in order to prevent the freeze we suggest to have at least 2GB of Ram and 30GB of Disk.

### Docker

Since dokploy runs with docker, is necessary to have docker installed on your server, but don't worry if you don't have docker installed, we will provide a command to automatically install docker on your server.




### Server

In order to use Dokploy, it is necessary to log in to your server, we suggest to use SSH.

There is guide to set up SSH on your server.

- [Hetzner SSH Configuration](https://community.hetzner.com/tutorials/add-ssh-key-to-your-hetzner-cloud)

If you are already logged in to your server. You can use the following command:

```bash
curl -sSL https://dokploy.com/install.sh | sh
```

This command will make the following things:

- Install Docker
- Initialize Docker Network
- Initialize Docker Swarm
- Install Dokploy

When the command is done, you can start using Dokploy.  

By simply type in the browser `your-ip-from-your-vps:3000` and you will be able to access Dokploy trought the internet.
