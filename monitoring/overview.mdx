Dokploy provides a monitoring system.


## Applications & Databases

You can monitor your applications or databases using the `Monitoring` tab.

If your application is running, it will display in realtime the CPU, Memory, Disk and Network usage, if not it will show you the last time it was running.


## Server

You can monitor your server usage using the `Monitoring` tab at `/dashboard/monitoring` route.

It will show you the CPU, Memory, Storage, Disk and Network usage of your server.
